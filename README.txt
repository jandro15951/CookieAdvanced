# Proyecto: Documentación de Archivos

## Estructura de Carpetas

- **assets/**: Contiene todos los archivos multimedia como imágenes, fuentes y otros recursos necesarios para las páginas HTML.

- **scripts/**: Contiene los archivos JavaScript utilizados para la funcionalidad de las páginas.
  - **Nota importante**: El archivo `onidframeverification.js` **no debe ser modificado**. Si es necesario realizar cambios, se deberá copiar el archivo y crear una nueva versión con el siguiente formato de nombre: `onidframeverification_v1_0_1.js`, incrementando el número de versión según sea necesario.

- **styles/**: Contiene los archivos CSS que definen los estilos de las páginas HTML.

## Archivos HTML

- **index.html**: Página principal del proyecto. Esta página sirve como punto de entrada.

- **steps.html**: Página de instrucciones o pasos que deben seguirse dentro del flujo de trabajo.

- **welcomepage.html**: Página de bienvenida para los usuarios.

## Notas adicionales

- Cada archivo tiene un propósito específico y se recomienda no hacer cambios sin antes validar las dependencias y el flujo del proyecto.
- Sigue las convenciones de nombrado y de versiones para mantener la consistencia y evitar conflictos.
